BRANDING

El Branding es el proceso de investigar, desarrollar y aplicar una característica distintiva o un conjunto de características a su organización para que los consumidores puedan comenzar a asociar su marca con sus productos o servicios.

FRONTEND

El desarrollo web front-end, también conocido como desarrollo web, es la práctica de producir HTML, CSS y JavaScript para un sitio web o una aplicación web para que un usuario pueda verlos e interactuar con ellos directamente.

DISEÑO UX/UI

El diseñador UX se ocupa de la situación y del problema en general. El diseñador UI se centra en los detalles, pule la interfaz a fondo y da los toques finales, responsables de hacer que la experiencia de usuario sea lo mejor posible.
